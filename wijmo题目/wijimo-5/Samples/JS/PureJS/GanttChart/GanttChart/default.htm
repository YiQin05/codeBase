GanttChart

This page shows how to create GanttChart using FlexChart.

Getting Started

Steps for getting started with the FlexChart to customize a GanttChart:

  1. Add references to Wijmo (see details here).
  2. Add a div element to serve as the chart host.
  3. Initialize the chart via JavaScript (set the chartType to 'Bar' and the itemsSource to the data).
  4. Add a series to the chart and set its binding property to the names of the fields that contain the start and end dates for each task.
  5. Customize AxisY to hide the major grid and show minor grid.

Dependencies & Percent Complete

Customize the chart by setting its itemFormatter property to a function that adds elements to show task dependencies and percent complete.
