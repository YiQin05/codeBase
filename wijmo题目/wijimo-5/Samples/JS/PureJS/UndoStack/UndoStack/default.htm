UndoStack

This demo shows an UndoStack class that can be used to provide form-level undo/redo functionality. It handles standard HTML elements, including text, check, and radio boxes, as well as Wijmo input and FlexGrid controls.

To see how it works, make changes to the form and then use the Undo/Redo buttons on the toolbar at the top of the page.
